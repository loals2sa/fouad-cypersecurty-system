💡 RECOMMENDED NEXT STEPS:

   1. View the assets:
      $ firefox PREVIEW.html
      (or open with any browser)

   2. Set a wallpaper immediately:
      $ gsettings set org.gnome.desktop.background picture-uri \
        file://$(pwd)/wallpapers/fouad_system_wallpaper_4k.png

   3. Copy logos to system:
      $ sudo mkdir -p /usr/share/pixmaps/fouad-system
      $ sudo cp logos/*.{png,svg} /usr/share/pixmaps/fouad-system/

   4. Read full installation guide:
      $ cat INSTALLATION_GUIDE.md
